# Educational Game: Digestive System Simulation
An interactive 2D educational simulation developed with Python and Pygame. This project uses game mechanics to teach users about the human digestive process, tracking food particles as they move through various biological stages.

## 🚀 Features
Interactive Learning: Players navigate food particles through the digestive tract, encountering different organ environments.

Real-time Physics: Implements collision detection and movement dynamics to simulate the mechanical flow of digestion.

Modular Levels: Organized into distinct stages (Mouth, Stomach, Intestines) representing different biological functions.

Educational UI: Visual feedback and progress tracking to reinforce biological concepts.

## 🛠️ Language
Language: Python 

Library: Pygame

Architecture: Object-Oriented Programming (OOP)

Utilizes class inheritance and encapsulation to model distinct biological entities.

Modular design allows for easy addition of new "levels" or organs.

## 🧠 Design Philosophy
This project was built with a focus on User-Centered Design. By translating abstract biological processes into a tangible, interactive format, the simulation lowers the barrier for students to understand human anatomy. The use of OOP ensures that the system is scalable, treating each organ as an object with its own unique properties and behaviors.
